Portfolio Projects

  • Yu-Gi-Search: A web application that allows users to search for cards from the popular TCG Yu-Gi-Oh, as well as create decks, comment on cards and browse other user's created decks. Made with HTML, CSS, JavaScript, Python/Flask, utilizing the Jinja2 templating engine.
  • Turtell: A Reddit clone built with minimal design in mind. Built using React/Redux and Python/Flask. Utilizing a PostgreSQL database linked with SQLAlchemy as the ORM.
  • Kilogram: Kilogram is a full-stack Instagram clone built with JavaScript, React/Redux and Python/Flask. Allows users to upload and organize media by hashtags, and browse content from users they follow
