Projects

Turtell

Turtell is a Reddit Clone built under two weeks. It allows you to create and join different communities, post in those communities for all members to view and comment on those posts to increase interaction between users. You can also search for communities using the searchbar built into the navbar that autocompletes search. I decided to not copy the styling of Reddit and work with my own design instead and was pretty satisfied with what I came up with. The goal was a simple minimalistic design that still showed off a good understanding of CSS and design fundamentals.

The frontend of this application utilizes React and Redux to ensure a smooth user experience with dynamically rendered components. The backend was created using Flask and SQLAlchemy with a Postgresql database, the API uses RESTFUL API conventions for intuitive queries.

Technologies used: JavaScript, React, Redux, Python, Flask, SQLAlchemy, PostgreSQL, HTML, CSS

Kilogram

Kilogram is a full-stack web application built through the collaborative efforts of four software engineers. It is a pixel-perfect clone of the infamous social media platform Instagram. The app allows users to upload media that can be organized by hashtags. User's posts can be shared publicly or with pre-approved followers. Users can also browse other users' content by tags. Lastly, users can like photos and follow other users to add their content to a personal feed.

I worked on Kilogram with Basel Hassan, Pawan Chahal, and Michael Eng. Built with JavaScript and Python, this web application uses React / Redux for the frontend and Flask for the backend, with SQLAlchemy as the ORM communicating with a PostgreSQL database. It also utilizes AWS to host images.

Technologies used: JavaScript, React, Redux, Python, Flask, SQLAlchemy, PostgreSQL, HTML, CSS, AWS

Flask Starter Application

The Flask Starter Application a github repository that allows people to quickly put together a CRUD application that uses Flask as a backend and serves server-side rendered dynamic pages. Using the Posts template feature provided within the repository, users can quickly spin up their own custom CRUD features incredibly fast.

This project serves mainly as a learning tool for people who are interested in getting into web development. The Flask Starter Application allows users to get used to a popular backend framework with as little boilerplate code as possible, with an example feature built out for them to look at as they build their own.

Technologies used: Python, Flask, SQLAlchemy, SQLite, HTML, CSS, Jinja
